# 📊 Análise de Falhas e Chamados na Rede

## Objetivo
Este projeto tem como objetivo analisar o desempenho operacional de uma rede de telecomunicações, com foco na identificação de padrões de falhas, eficiência das equipes e tempo de resolução de chamados.

---

## Contexto
No setor de telecomunicações, a análise de chamados é essencial para garantir qualidade de serviço e eficiência operacional.  
A partir disso, este projeto busca transformar dados operacionais em insights para apoio à tomada de decisões necessárias.

---

## Ferramentas utilizadas
- SQL (T-SQL)
- Para execução online, foi utilizada adaptação para SQLite.
- Banco de dados relacional
- GitHub (documentação e versionamento)

---

## Base de dados
A base de dados simula um ambiente real de telecom, contendo informações como:
- Região
- Equipe responsável
- Data de abertura e fechamento
- Status do chamado

---

## Análises realizadas

### Regiões com mais problemas
Identificação das regiões com maior volume de chamados, permitindo detectar possíveis gargalos na infraestrutura.

---

### Tempo médio de resolução
Cálculo do tempo médio entre abertura e fechamento dos chamados, auxiliando na avaliação de eficiência operacional.

---

### Equipes mais eficientes
Análise do desempenho das equipes com base no tempo médio de resolução, destacando aquelas com maior eficiência.

---

## Principais insights
- Regiões com maior volume de chamados indicam possíveis pontos críticos da rede  
- Diferença no tempo de resolução entre equipes evidencia oportunidades de melhoria  
- Monitorar tempo médio de atendimento contribui para melhoria de SLA  

---

## Habilidades desenvolvidas
- SQL avançado (JOIN, GROUP BY, agregações)
- Análise de dados operacionais
- Modelagem de Dados
- Identificação de padrões e geração de insights
- Pensamento analítico aplicado ao negócio

---

## Tomada de decisão
Com base nas análises realizadas, é possível:

- Priorizar investimentos em regiões com maior volume de falhas  
- Reavaliar processos operacionais das equipes com menor desempenho  
- Definir estratégias para redução do tempo médio de atendimento  
- Melhorar a qualidade do serviço e a satisfação do cliente
